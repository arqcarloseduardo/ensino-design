# RITMO: A CADÊNCIA DA EXPERIÊNCIA

Na gramática visual de Christian Leborg, ritmo é a repetição organizada de elementos no espaço. Ele cria cadência visual, como na música.

👉 **Se a hierarquia diz o que vem primeiro, o ritmo diz como isso se desenrola no tempo.**

## 🧠 Psicologia (Jung)
Sob a ótica de Carl Jung:
* O ritmo se conecta a padrões profundos da psique: batimentos, respiração e ciclos naturais.
* **Ele ativa uma sensação de:**
    * **Previsibilidade** → segurança
    * **Quebra de padrão** → atenção

👉 **Ritmo é ordem percebida ao longo do tempo.**

## 👁️ Percepção e Gramática
O ritmo acontece através de:
* **Repetição** → elementos iguais.
* **Alternância** → variação controlada.
* **Progressão** → mudança gradual (escala, cor, forma).
* **Espaçamento** → define a cadência.

**Tipos comuns:**
* **Regular** → estável.
* **Irregular** → dinâmico.
* **Progressivo** → evolutivo.

👉 **Ele cria:** Fluxo visual, Continuidade e Expectativa.

## ⚛️ Estrutura (Design Atômico)
Com Brad Frost:
* **Ritmo aparece na repetição de:**
    * Componentes
    * Espaçamentos (*spacing system*)
    * Padrões de layout

👉 É o que dá consistência sistêmica e escalabilidade.

## ❤️ Emoção + Função
Segundo Don Norman:
* **Visceral** → sensação de harmonia ou tensão.
* **Comportamental** → facilita a escaneabilidade.
* **Reflexivo** → transmite sofisticação ou caos.

> **Ritmo regular** → calma, controle
> **Ritmo quebrado** → energia, surpresa

## 🧩 Síntese Nexialista
Ritmo é:
* **Psicologicamente** → ciclo
* **Visualmente** → cadência
* **Estruturalmente** → repetição
* **Emocionalmente** → fluidez
