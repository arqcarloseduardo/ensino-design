# 🧩 O QUE É COMPOSIÇÃO (ALÉM DO ÓBVIO)

Composição é o campo onde tudo isso acontece.
Mas, dentro da abordagem nexialista, ela não é só “organizar elementos” — é orquestrar significado.

Na lógica de Christian Leborg:
* **Composição = relações entre elementos**
* Não é o ponto, linha ou cor isolados
* É como eles interagem no espaço

👉 **Ou seja:** Design começa de verdade aqui.

## 🧠 PSICOLOGIA (JUNG)
Com Carl Jung:
* A composição organiza a experiência interna
* **Ela pode criar:**
    * **Ordem** → segurança
    * **Caos** → tensão
    * **Equilíbrio** → harmonia

👉 A composição é a tradução visual de um estado psíquico.

## 👁️ GRAMÁTICA VISUAL (AS RELAÇÕES)
Aqui entram os princípios:
* Contraste
* Hierarquia
* Ritmo
* Equilíbrio
* Escala
* Proximidade

👉 Esses são os **“verbos”** do design.

**Exemplo simples:**
* Um ponto sozinho → presença
* Vários pontos alinhados → ritmo
* Um maior que outro → hierarquia

## ⚛️ ESTRUTURA (DESIGN ATÔMICO)
Com Brad Frost:
* **Composição = organização de moléculas e organismos**
* Define como o sistema se comporta

👉 Aqui ela deixa de ser estética e vira arquitetura.

## ❤️ EMOÇÃO + FUNÇÃO
Segundo Don Norman:
* **Visceral** → impacto visual geral
* **Comportamental** → usabilidade (fluxo, leitura)
* **Reflexivo** → significado percebido

👉 Uma má composição confunde.
👉 Uma boa composição guia sem esforço.

## 🧩 SÍNTESE NEXIALISTA
Composição é:
* **Psicologicamente** → organização da experiência
* **Visualmente** → relações
* **Estruturalmente** → arquitetura
* **Emocionalmente** → narrativa silenciosa
