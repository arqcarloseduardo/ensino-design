# ALINHAMENTO: A ORDEM INVISÍVEL

Na gramática visual de Christian Leborg, alinhamento é a organização dos elementos ao longo de eixos comuns. Ele cria uma linha estrutural invisível que conecta tudo.

👉 **O alinhamento responde: “Como isso se sustenta como um todo coerente?”**

## 🧠 Psicologia (Jung)
Sob a ótica de Carl Jung:
* A mente busca ordem e continuidade para processar a realidade.
* **O alinhamento reduz o esforço cognitivo.**
* Ele se conecta à necessidade psíquica de **coerência e integração lógica**.

👉 **Quando algo não está alinhado, sentimos um “incômodo” inconsciente mesmo sem saber explicar.**

## 👁️ Percepção e Gramática
O alinhamento cria organização invisível, unidade visual e fluxo consistente.



**Tipos comuns:**
* **Alinhamento à esquerda** → Natural e confortável (leitura ocidental).
* **Centralizado** → Formal, equilibrado e estático.
* **À direita** → Menos comum, usado para diferenciação ou expressividade.
* **Em grid** → Altamente estruturado e matemático.

👉 **Regra prática:** Nada deve estar “solto” no layout; cada elemento deve ter uma relação visual com outro.

## ⚛️ Estrutura (Design Atômico)
Com Brad Frost:
* **Alinhamento é sustentado por:**
    * Grids.
    * Colunas.
    * Sistemas de layout fundamentais.

👉 É o que garante a consistência técnica entre diferentes telas e componentes de um sistema.

## ❤️ Emoção + Função
Segundo Don Norman:
* **Visceral** → sensação imediata de ordem ou caos.
* **Comportamental** → melhora drasticamente a escaneabilidade.
* **Reflexivo** → transmite profissionalismo e cuidado.

> **Bom alinhamento** → gera confiança.
> **Mau alinhamento** → transmite amadorismo.

## 🧩 Síntese Nexialista
Alinhamento é:
* **Psicologicamente** → coerência
* **Visualmente** → ordem invisível
* **Estruturalmente** → grid
* **Emocionalmente** → confiança
