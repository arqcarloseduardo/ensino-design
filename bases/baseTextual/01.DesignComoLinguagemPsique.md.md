# O DESIGN COMO LINGUAGEM DA PSIQUE

Quando trazemos Carl Jung para o design, saímos do nível estético e entramos no simbólico.
Jung nos mostra que existem arquétipos — padrões universais que habitam o inconsciente coletivo. No design, isso se traduz em:

* **O herói** → marcas de superação (Nike, por exemplo)
* **O cuidador** → estética acolhedora (área da saúde)
* **O sábio** → minimalismo, clareza, autoridade

👉 Um designer nexialista não escolhe cores ou formas apenas por gosto — **ele evoca estados psicológicos.**

## 🧠 GRAMÁTICA VISUAL COMO SINTAXE DO SENTIDO
Christian Leborg propõe algo essencial: o design é uma linguagem com gramática própria.

**Elementos básicos:**
* Ponto
* Linha
* Forma
* Cor
* Textura

Mas o mais importante não são os elementos — são as **relações**:
* **Contraste** → tensão
* **Repetição** → ritmo
* **Escala** → hierarquia
* **Direção** → narrativa visual

👉 Aqui começa a ponte com Jung:
* A gramática visual é o **como**.
* A psicologia é o **porquê**.

## ⚛️ DESIGN ATÔMICO COMO ESTRUTURA DO PENSAMENTO
Brad Frost introduz o conceito de design atômico:

1.  **Átomos** → botões, cores, tipografia
2.  **Moléculas** → combinações simples
3.  **Organismos** → blocos funcionais
4.  **Templates** → estrutura
5.  **Páginas** → experiência final

Mas indo além do digital:
👉 **Isso é uma filosofia estrutural**
Você não está desenhando telas — está construindo sistemas coerentes de significado.

## ❤️ DESIGN EMOCIONAL + ⚙️ FUNCIONAL
Aqui entramos na tensão mais importante:

* **Funcional** → resolve um problema
* **Emocional** → cria vínculo

Don Norman fala sobre três níveis:
* **Visceral** → impacto imediato (beleza)
* **Comportamental** → usabilidade
* **Reflexivo** → significado / memória

👉 Um bom design funciona.
👉 Um grande design permanece.
